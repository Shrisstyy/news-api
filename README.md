# news-api
A real-time news aggregation system that fetches the latest news articles based on user interest using the NewsAPI.

Features:

1. Search news by topic or keyword
2. Fetch latest articles in real-time
3. Displays article title and source link
4. Easy to use command-line interface

Technologies Used:

1.Python
2.Requests Library
3.NewsAPI

Requirements:

Before running this project, make sure you have:

1. Python 3.8+
2. requests library installed
3. NewsAPI API Key
